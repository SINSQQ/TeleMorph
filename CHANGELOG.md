## [1.0.0] — 2026-06-28

### Added
- Initial public release of **TeleMorph**.
- Unified API that wraps **TGConvertor** and **opentele2**.
- Conversion between **Telethon**, **Pyrogram**, **Kurigram** and **Tdata**.
- Device randomizer with 14+ real device profiles (Samsung, Pixel, Xiaomi, OnePlus, Huawei, iPhone, Desktop).
- CLI (`telemorph`) supporting `list-formats`, `list-api-types`, `info`, `convert`, `batch-convert`.
- Async + sync API surface.
- Optional extras: `[telethon]`, `[pyrogram]`, `[kurigram]`, `[opentele]`, `[all]`, `[dev]`.
- Documentation site scaffold under `docs/`.
